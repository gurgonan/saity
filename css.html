CSS

CSS (Cascading Style Sheets) — формальный язык декорирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML). Основная цель разработки CSS — отделение описания логической структуры веб-страницы от описания её внешнего вида. Такое разделение позволяет увеличить доступность документа, предоставить большую гибкость и возможность управления его представлением, а также уменьшить сложность и повторяемость в структурном содержимом. Кроме того, CSS позволяет представлять один и тот же документ в различных стилях или методах вывода.

Как работают таблицы стилей

  • Начните с документа, размеченного на языке HTML.
  • Напишите правила стилей, чтобы показать, какими вы хотели бы видеть определенные элементы.
  • Присоедините правила стилей к документу. Когда браузер отображает документ, он следует вашим правилам представления элементов

Написание правил

Таблица стилей составлена из одной и более инструкций (называемых правилами или наборами правил), которые описывают, как элемент или группа элементов должны отображаться. Первый шаг в изучении CSS — это знакомство с частями правил. Каждое правило выбирает элемент и объявляет, как он должен выглядеть.
Следующий пример содержит два правила. Первое из них задает отображение всех элементов h1 в документе зеленым цветом; второе — определяет, что шрифт абзацев должен быть мелким без засечек.
h1 { color: green; }
р { font-size: small; font-family: sans-serif; }
Согласно терминологии CSS существует две главные части правила — это селектор, устанавливающий элемент или элементы, на которые надо воздействовать, и определение, предоставляющее инструкции представления. Определение, в свою очередь, составлено из свойства (например, color) и его значения (green ), разделенных двоеточием и пробелом. Одно или более определений размещаются внутри фигурных скобок.

Селекторы

В предыдущем небольшом примере элементы h1 и р выступают как селекторы. Этот наиболее часто используемый тип селектора называется селектор элемента. Свойства, определенные для каждого селектора, будут соответственно применяться ко всем элементам h1 и р в документе.

Определения

Определение составлено из нары свойство/значение. Допустимо использовать несколько определений в одном правиле, например указанное выше правило для элемента р имеет свойства как font-size , так и font-family .
Каждое определение должно оканчиваться точкой с занятой для отделения его от следующего

Присоединение таблиц к документу

Есть три способа применить информацию о стиле к HTML-документу.

Внешние таблицы стилей

Внешняя таблица стилей — это отдельный текстовый (и только текстовый) документ, который содержит ряд правил стилей. Он должен иметь имя с расширением .css. Документ с расширением .css связывается или импортируется в один или более HTML-документ

Глобальные таблицы стилей

Такая таблица размещается в документе при помощи элемента style и ее правила применяются только в этом документе. Элемент style должен быть расположен в разделе head документа.

Встроенные таблицы стилей

Вы можете применять свойства и значения к одному элементу, используя в нем атрибут style, как показано ниже:
<hl style="color: red">Ведение</hl>

Сгруппированные селекторы

Если вам когда-нибудь понадобится применить одно и то же свойство к нескольким элементам, можете сгруппировать селекторы в одно правило, разделив их запятыми.Пример:
hi, h2, р, em, img { border: lpx solid blue; }

Свойства шрифта

В CSS шрифт задается при помощи набора связанных с ним свойств для указания вида шрифта, размера, насыщенности и начертания.

Указание имени шрифта

Используйте свойство font-family для задания шрифта или списка шрифтов

Определение размера шрифта

Чтобы задать размер текста, используйте свойство с именем font-size.
  • Можно использовать одну из единиц измерения длины в CSS
    h1 { font-size: 1.5em; }
  • Используя одно из абсолютных зарезервированных слов (xx-small, x-small, small, medium, large, x-large, xx-large).
    • h1 { font-size: x-large; }
  • Используя относительные (или сравнительные) зарезервированные слова (larger или smaller), чтобы заставить отображаться текст больше или меньше близлежащего текста.
    strong { font-size: larger; }

Насыщенность шрифта

Если вы хотите, чтобы текстовый элемент отображался шрифтом с полужирным начертанием, используйте свойство font-weight.
Принимаемые значения, normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 |900 | inherit

Капитель

Это отдельный дизайн шрифта, который использует уменьшенные заглавные буквы вместо строчных. Свойство font-variant предназначено для того, чтобы позволить задавать подобный капительный шрифт для текстовых элементов.

Изменение цвета текста

Цвет текста меняется при помощи свойства color.Его значением может быть предопределенное имя цвета или числовое значение, описывающее конкретный оттенок в цветовой модели RGB.
h1 { color: green; }

Высота строки

Свойство line-height определяет минимальное расстояние между базовыми линиями строк текста.
р { line-height: 2em; }

Отступы

Свойство text-indent задает отступ первой строки текста определенной ширины
р { text-indent: 2em; }

Горизонтальное выравнивание

Вы можете выравнивать текст на веб-страницах при помощи свойства text-align
text-align: left выравнивает текст по левому краю
text-align: right выравнивает текст по правому краю
text-align: center выравнивает текст по центру в текстовом блоке
text-align: justify выравнивает текст и по правому и левому краям (по ширине)

Подчеркивания и другие декоративные эффекты

Свойство text decoration нужно для того, чтобы разместить линию под или над текстом, зачеркнуть его или отключить подчеркивание ссылок.
text-decoration: underline подчеркивает элемент
text-decoration: overline рисует линию над текстом
text-decoration: line-through рисует линию через текст
text-decoration: blink делает текст мерцающим
использование свойства text-decoration для отключения линии, которая появляется автоматически под текстом ссылки, как показано здесь:
p { text-decoration: none; }
